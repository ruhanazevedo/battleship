# Contributing

Bug reports and pull requests are welcome.

## Getting started

```bash
git clone https://github.com/ruhanazevedo/battleship.git
cd battleship
mvn package -q
java -jar target/battleship.jar
```

## Before submitting a PR

- Run `mvn test` and make sure everything passes
- Keep changes focused — one thing per PR
- Write a clear commit message explaining what changed and why

## Reporting bugs

Open an issue with:
- What you expected to happen
- What actually happened
- Steps to reproduce
- Your Java version and OS
